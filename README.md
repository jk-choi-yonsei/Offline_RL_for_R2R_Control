# Drift-Aware Offline Reinforcement Learning for Run-to-Run Control in Chemical Mechanical Planarization

## Overview

This repository contains the code, pre-trained checkpoints, and evaluation results for **SARC** (State-Adaptive R2R Controller), an offline reinforcement learning framework for CMP run-to-run control under progressive equipment wear.

SARC comprises:
1. **Drift Encoder** — a bottleneck MLP pre-trained via auxiliary RR-prediction to distill wear telemetry into a 2-D context vector
2. **Drift-conditioned CQL policy** — maps (state, context) to zone-pressure recipes using Conservative Q-Learning

## Reproducing Paper Results

### Using pre-trained checkpoints (recommended)

All paper-final checkpoints are in `results/checkpoints/`. To evaluate without re-training:

```bash
# CMP2 (Table III in paper)
python scripts/run_cmp2_final.py

# CMP1 (Table II in paper)
python scripts/run_unified_phase1.py

# Aggregate and compute statistics
python scripts/aggregate_5seed.py
python scripts/wilcoxon_holm.py
python scripts/final_statistics.py
```

## Checkpoint Naming Convention

| Pattern | Description |
|---------|-------------|
| `sarc_model_final_cmp2_cql1.0_s{seed}.pt` | SARC (CMP2, seed) |
| `sarc_no_drift_final_cmp2_cql1.0_s{seed}.pt` | SARC-no-drift ablation (CMP2) |
| `bc_model_final_cmp2_cql1.0_s{seed}.pt` | BC (CMP2) |
| `sarc_cmp1_model_unified_s{seed}_s{seed}.pt` | SARC (CMP1, chronological split) |
| `sarc_sim_{mild,medium}_cql1.0_s{seed}_model.pt` | SARC (Simulator) |
| `world_model_{cmp1,cmp2,sim_medium}_best.pt` | Neural World Model (5-ensemble) |

`*_best.pt` suffix = val-MAE best checkpoint (used for paper results).
